# CoinAssets

Отслеживание цен криптовалют в режиме реального времени с испльзованием сервиса https://docs.coincap.io/

Стек: Swift, UIKit, AutoLayout, Alamofire, CoreData, TableKit

Архитектура: Clean Architecture, VIP
